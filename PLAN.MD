# Project Plan - Zoloti Maky Landing Page

## Executive Summary
Building a professional landing page for Zoloti Maky dance group to showcase their work, attract new members, and enable inquiries from potential clients.

## Technical Approach

### Technology Decision
**Recommended Stack: HTML/CSS/JavaScript (Static Site)**
- **Why:** Simple, fast, no server needed, easy to host anywhere
- Perfect for a landing page with form submission
- Easy for me to update content for you in the future
- Can use free hosting (Netlify, Vercel, GitHub Pages)
- Excellent performance and SEO
- Alternative considered: React/Next.js (overkill for this scope)

### Form Handling
- Use a form service like Formspree or Web3Forms (free tier available)
- Alternative: EmailJS for direct email submissions
- No backend server needed

## Hosting Options (Recommendations)

### Option 1: Netlify (Recommended)
- **Cost:** Free
- **Pros:** Easy deployment, fast, custom domain support, form handling included
- **Best for:** This project

### Option 2: Vercel
- **Cost:** Free
- **Pros:** Similar to Netlify, very fast
- **Best for:** Alternative to Netlify

### Option 3: GitHub Pages
- **Cost:** Free
- **Pros:** Simple, reliable
- **Cons:** No built-in form handling
